# Interview Project Stories

## Retail KPI Dashboard
I cleaned retail order data and created KPI tables by month, category, region, and channel. I tracked revenue, profit, orders, customers, units sold, average order value, profit margin, and return rate.

## ETL Data Quality Workflow
I built an ETL script that checks duplicates, missing emails, missing amounts, invalid records, and inconsistent formats. It outputs a clean transaction dataset and a quality report.

## Stock Price Prediction using LSTM
I built a time-series forecasting workflow using stock close prices. I scaled the data, created rolling 60-day sequences, trained an LSTM, and compared actual versus predicted prices.

## Customer Churn EDA and ML Preparation
I explored churn data, checked missing values, calculated churn rate, prepared numeric and categorical features, and trained a baseline model.

These are personal portfolio projects created to demonstrate my analytics workflow. I can explain the data, code, outputs, and what I learned.
