# Retail KPI Dashboard

## Goal
Build dashboard-ready KPI datasets from raw retail order data.

## What this proves
- Data cleaning
- KPI creation
- Business performance reporting
- Power BI/Tableau/Excel dashboard preparation

## KPIs
Revenue, profit, profit margin, orders, customers, units sold, average order value, return rate, monthly trend, category performance, region performance, and channel performance.

## How to run
```bash
pip install pandas
python scripts/build_dashboard_dataset.py
```

## Interview explanation
I cleaned raw retail order data, filtered completed sales, created monthly/category/region/channel KPI tables, and prepared the output for dashboarding. The project shows how I translate transactional data into business performance insights.
