# Customer Churn EDA and ML Preparation

## Goal
Perform exploratory data analysis and prepare customer data for machine learning.

## What this proves
- EDA
- Missing value checks
- Feature preparation
- Encoding categorical variables
- Scaling numeric variables
- Baseline classification

## How to run
```bash
pip install pandas scikit-learn
python scripts/eda_ml_prep.py
```

## Interview explanation
I explored customer churn data, checked data quality, calculated churn rate, prepared numeric and categorical features, and trained a logistic regression baseline model. This shows how I move from raw data to model-ready data.
