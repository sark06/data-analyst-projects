# Data Analyst Portfolio Projects

This repository contains personal Data Analyst portfolio projects built to demonstrate practical skills in data cleaning, ETL workflows, KPI reporting, exploratory data analysis, machine learning preparation, and time-series forecasting.

These projects use synthetic but realistic datasets and are designed to show end-to-end analytics workflows using Python, Pandas, Scikit-learn, TensorFlow, and dashboard-ready data outputs.

---

## Projects

### 1. ETL Data Quality Workflow

**Link:** [View Project](./02_etl_data_quality_workflow)

Created an ETL workflow to clean messy transaction data, remove duplicates, handle missing values, standardise formats, convert currency values, and generate a data quality report for reliable reporting.

**Skills:** Python, Pandas, ETL, Data Validation, Data Quality, Automation

---

### 2. Retail KPI Dashboard

**Link:** [View Project](./01_retail_kpi_dashboard)

Built a dashboard-ready retail analytics project using order-level data. Cleaned raw sales data, created monthly, category, region, and channel KPI tables, and prepared outputs for Power BI, Tableau, or Excel reporting.

**Skills:** Python, Pandas, KPI Reporting, Data Cleaning, Dashboard Preparation

---

### 3. Customer Churn EDA and ML Preparation

**Link:** [View Project](./04_customer_churn_eda_ml_prep)

Performed exploratory data analysis on customer churn data, checked missing values, calculated churn rate, prepared numeric and categorical features, and trained a logistic regression baseline model.

**Skills:** Python, Pandas, Scikit-learn, EDA, Feature Preparation, Classification

---

### 4. Stock Price Prediction using LSTM

**Link:** [View Project](./03_stock_price_prediction_lstm)

Built a time-series forecasting project using stock price data. Prepared rolling 60-day sequences, scaled closing prices, trained an LSTM model, and compared actual versus predicted stock prices.

**Skills:** Python, TensorFlow, LSTM, Time-Series Forecasting, Machine Learning

## How to Run

Clone the repository:

```bash
git clone https://github.com/sark06/data-analyst-projects.git
cd data-analyst-projects
```

Install common packages:

```bash
pip install pandas numpy matplotlib scikit-learn
```

For the LSTM project, install TensorFlow:

```bash
pip install tensorflow
```

Each project has its own README file with project-specific instructions.
